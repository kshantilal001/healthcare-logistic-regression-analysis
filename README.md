# healthcare-logistic-regression-analysis
# Healthcare Data Analysis using Logistic Regression (BI-RADS Dataset)

## Overview
This project analyses a healthcare dataset to identify factors associated with breast cancer malignancy using statistical modelling techniques.

The objective is to understand relationships between diagnostic variables and improve prediction accuracy.

---

## Tools & Technologies
- Minitab  
- Statistical Methods:  
  - Chi-Square Test  
  - Binary Logistic Regression  

---

## Dataset
- BI-RADS dataset (Breast Imaging Reporting and Data System)
- Includes:
  - Age  
  - Tumour characteristics  
  - Diagnostic scores  
  - Malignancy outcome  

---

## Key Steps

### 1. Data Cleaning
- Identified ~16% missing values in the dataset  
- Evaluated potential impact of missing data on analysis  
- Prepared dataset for modelling  

### 2. Exploratory Analysis
- Analysed relationships between variables and malignancy  
- Used contingency tables to understand distributions  

### 3. Statistical Testing
- Performed Chi-Square tests to identify significant relationships  

### 4. Logistic Regression Modelling
- Built a binary logistic regression model  
- Interpreted odds ratios for key predictors  
- Evaluated model performance using classification metrics  

---

## Key Findings
- Higher BI-RADS scores strongly associated with malignancy  
- Age and tumour characteristics significantly influenced outcomes  
- Model achieved strong predictive performance (~91% concordance)
